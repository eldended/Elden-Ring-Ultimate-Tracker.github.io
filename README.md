# Elden-Ring-Ultimate-Tracker.github.io
Интерактивный веб-интерфейс для отслеживания прогресса прохождения Elden Ring. Сайт позволяет отмечать убитых боссов (сохранение прогресса в LocalStorage), быстро находить информацию о локациях, рекомендуемом уровне и слабостях противников через систему навигации. Полностью адаптивный дизайн в стиле темного фэнтези.
